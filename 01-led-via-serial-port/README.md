# 01 - Blinking leds by user input

A simple arduino program that blinks two leds based on the user input. Using`Serial.parseInt` to read the values entered. 

## Schematics

![schematics](schematics.png)

## Components

- Arduino Uno R3

- Breadboard

- 6x M-M wires

- 1x Red LED

- 1x Yellow LED

- 2x 220 ohm resistors
